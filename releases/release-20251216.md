# release-20251216

Release type: Multiplayer alpha

Release date: 16 December 2025

Release post: [Bugfix release for multiplayer alpha]({% link news/_posts/2025-12-16-multiplayer-alpha-bugfix.md %})

GitHub release: [release-20251216](https://github.com/OpenE2140/OpenE2140/releases/release-20251216)

## What's new
- added an upgraded version of Power Plant with two extra modules
    (a temporary solution until building upgrades are implemented)
- fixed crashes, visual bugs and other issues related to Bantha and Heavy Lifters
- other small fixes
- enabled production slot hotkeys
- Mine no longer produces ore crates too quickly
- Debug Menu is now disabled by default
- Defense Tower now requires only Bio-Center to build (like in E2140)
- light vehicles can no longer crush infantry

For full changelog, see the [release page](https://github.com/OpenE2140/OpenE2140/releases/release-20251216) on GitHub.


## Known issues
- No playable AI in Skirmish: Skirmish mode is available, but there is no AI opponent yet.
- Single-player campaigns not implemented.
- Occasional crashes or visual glitches may occur in some situations.
- Some UI elements and features are still being polished.

For a detailed, up-to-date list of known issues and their status, see the [Known issues page]({% link known-issues.md %}).
